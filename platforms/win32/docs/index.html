Win32 Squeak

This page contains information and downloads for the Win32 Squeak virtual machine.

Information

Specific information about the Win32 VM is available here:

  • Building Your Product
    NEW! A complete tutorial for building a custom VM and packaging your product.
  • Compiling the Win32 VM
    This document discusses the steps to build a Win32 VM for yourself.
  • Win32 Ini File Settings
    A description of the various settings in INI file utilized by the VM.
  • Win32 VM preferences
    A description of the preferences accessible via the VM's preferences menu.
General information about the Squeak VM is available here:
  • A Step-by-Step guide to create a plugin on Windows
    An updated set of instructions as of december 2008.
  • A Tour of the Squeak Object Engine
    A must read for the serious VM hacker, it outlines the fundamentals of Squeak's virtual machine.
  • Extending Squeak by Writing Plugins
    Goes through the process of writing plugins for the Squeak VM, using Spread as an example.
  • VMMaker documentation
    Information about VMMaker the tool to generate the Squeak virtual machine directly from Squeak.
